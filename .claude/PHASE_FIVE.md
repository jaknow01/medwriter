# PHASE FIVE of Development (out of MVP)

At this phase I would like to extend the capabilities of the system by adding a RAG functionality. The idea is that the user can add their own PDFs that will serve as additional knowledge source for the agent. This requires a vector store and additional functionalities in the backend, as well as a way to store these documents somewhere. 
In this phase don't implement anything yet. Instead analyse the system (assuming it is working as intended up to this point) and assess how this RAG functionality could be integrated in the best way engineering-wise and in a way that maintains scalability of the system.
